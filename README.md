# Homeassistant im Schwanen
## Übersicht
Im Jahr 2025 wurde im Schwanen ein Homeassistant installiert. Ziel ist es in erster Linie die Heizunssteuerung zu optimieren.
Dazu werden alle Heizkörper mit intelligenten Thermostaten ausgestattet.
